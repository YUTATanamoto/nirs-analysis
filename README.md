## Usage  
Before you start, you have to install 'Python', 'pip' and 'git'.  
```
$ git clone https://github.com/YUTATanamoto/nirs-analysis.git  
$ cd nirs-analysis  
$ pip install pipenv  # if you've allready installed pipenv, you can skip this command.  
$ pipenv install  
$ pipenv shell  
$ jupyter notebook
```
